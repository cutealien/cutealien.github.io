4.5 Criterios de convergencia

Para determinar la convergencia de las series trabajadas hasta ahora, se ha considerado las primeras sumas parciales $S_n$ y después ver si tienen límite finito cuando $n \to \infty$. Sin embargo, no siempre es posible encontrar una expresión para $S_n$ de manera que resulte sencillo evaluar el límite cuando $n \to \infty$, lo que hace difícil establecer la convergencia de la serie.

Es posible establecer la convergencia o divergencia de una serie sin tener conocimiento explícito de las sumas parciales, para esto se han desarrollado criterios de convergencia. El primer criterio que se da, es una condición necesaria para la convergencia.

Teorema 4.5.1. Si la serie $\sum_{n=0}^\infty a_n$ converge, el $n$-ésimo término tiende a cero.

Demostración.

Sea $S_n = a_1 + a_2 + \cdots a_n$, entonces $a_n = S_n - S_{n-1}$. Cuando $n \to \infty$, $S_n$ y $S_{n-1}$ tienen el mismo límite y por lo tanto, $a_n \to 0.$

Criterios de convergencia para series de términos no negativos.

Teorema 4.5.2. Si $a_n \geq 0 $ para cada $n geq 1$, la serie $\sum_{n=0}^\infty a_n$ converge si y sólo si la sucesión de sumas parciales está acotada superiormente.

Demostración.

Puesto que las sumas parciales son monótonas crecientes, se aplica el Teorema 4.2.3 para obtener la condición necesaria y suficiente de convergencia. Recuerde que una serie es una sucesion $\{S_n \}$ donde $S_n = a_1 + a_2 + \cdots + a_n$.
