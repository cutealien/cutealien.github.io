Fórmulas básicas de integración
$$\int x^n\,dx=\frac{x^{n+1}}{n+1}+C \qquad \mbox{ con }n\in\mathbf{N} \setminus\{-1\} \mbox{ y }C\mbox{ una constante}.$$
$$\int e^{kx}\,dx=\frac{e^{kx}}{k}+C \qquad \mbox{con }C \mbox{ una constante.}$$
$$\int\frac{1}{x}\,dx=\ln x+C \qquad \mbox{con }C \mbox{ una constante.}$$
$$\int \cos x\,dx=\sin x+C \qquad \mbox{con }C \mbox{ una constante.}$$
$$\int \sin x\,dx=-\cos x+C \qquad \mbox{con }C \mbox{ una constante.}$$
\begin{split} \frac{d}{dx}\tan x &=\sec^2 x \Rightarrow \int\sec^2 x\, dx=\tan x +C \\ \frac{d}{dx}\cot x &=-\csc^2 x \Rightarrow \int\csc^2 x\, dx=-\cot x +C \\ \frac{d}{dx}\sec x &=\sec x \tan x \Rightarrow \int\sec x \tan x\, dx=\sec x +C \\ \frac{d}{dx}\csc x &=-\csc x \cot x \Rightarrow \int\csc x \cot x\, dx=-\csc x +C \end{split}
\begin{split} \frac{d}{dx}\arcsin x &=\frac{1}{\sqrt{1-x^2}} \Rightarrow \int \frac{1}{\sqrt{1-x^2}} \, dx=\arcsin x +C \\ \frac{d}{dx}\arccos x &=-\frac{1}{\sqrt{1-x^2}} \Rightarrow \int \frac{1}{\sqrt{1-x^2}} \, dx=-\arccos x +C \\ \frac{d}{dx}\arctan x &=\frac{1}{1+x^2} \Rightarrow \int \frac{1}{1+x^2} \, dx=\arctan x +C \end{split}
